Requirements:
- Browser (Chrome, Edge, Firefox).
- VS Code + “Live Server” extension.
	 
Installation:
1. Create a project folder, for example: C:\Users\YOUR_USER\Documents\NasaAPP
2. Put BOTH files inside: cities.db and index.html
3. Install the Live Server extension (VS Code)
	a) Open VS Code.
	b) Press Ctrl+Shift+X (Extensions view).
	c) Search: Live Server (author: Ritwick Dey).
	d) Click “Install”.
	e) If prompted, click “Reload” to activate the extension.
4.	Start the site with Live Server (recommended)
	a) In VS Code, go to File > Open Folder… and select your project folder (e.g., ...\NasaAPP\).
	b) In the Explorer, right-click on index.html.
	c) Click “Open with Live Server”.
	d) Your browser will open a URL like: http://127.0.0.1:5500/index.html
	e) You should see the city selector, charts.
